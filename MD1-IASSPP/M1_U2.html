Módulo 1 – Unidade 2

Tipos de IA e impactos no trabalho técnico

Duração: 4 horas

Por que estudar os tipos de IA?

Nem toda IA faz a mesma coisa.

Cada tipo automatiza uma parte diferente do trabalho humano.

Evolução da IA

  • Sistemas baseados em regras
  • Machine Learning
  • Deep Learning
  • Redes Neurais
  • Transformers
  • IA Generativa

Machine Learning

Sistemas que aprendem padrões a partir de dados.

Exemplo: filtro de spam, recomendação da Netflix.

Deep Learning

Um tipo de Machine Learning baseado em redes neurais profundas.

  • Trabalha bem com imagens, voz e linguagem
  • Exige muitos dados

O que é uma Rede Neural?

Modelo inspirado no cérebro humano.

Conecta "neurônios artificiais" que aprendem padrões.

Tipos de Redes Neurais

  • Redes Neurais Densas
  • Redes Neurais Convolucionais (CNNs)
  • Transformers

CNNs

Especializadas em imagens e visão computacional.

Usadas em reconhecimento facial, carros autônomos e diagnósticos médicos.

Transformers

Arquitetura focada em linguagem.

  • Base dos LLMs
  • Permite entender contexto

Transformers e IA Generativa

ChatGPT, Copilot e Gemini são baseados em Transformers.

Do modelo simples ao complexo

  • Filtro de spam → ML simples
  • Reconhecimento de imagem → CNN
  • Chatbots e código → Transformers

Impacto no trabalho técnico

  • Classificação automática
  • Revisão de código
  • Geração de testes
  • Correção de bugs

Ferramentas baseadas em IA

  • GitHub Copilot
  • ChatGPT
  • Code Review automático

O novo programador

Ele trabalha com IA, não contra ela.

Soft skills que se tornam essenciais

  • Pensamento crítico
  • Capacidade de validar
  • Comunicação
  • Aprender continuamente

Atividade

Cite uma ferramenta de IA e identifique qual tipo de modelo ela usa.

Encerramento

Quem entende os tipos de IA sabe usá-los melhor.
