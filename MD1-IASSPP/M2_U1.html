Soft Skills para Programadores

IA como assistente de desenvolvimento

Duração: 4 horas

Objetivo da Unidade

  • Entender o papel da IA no dia a dia do programador
  • Usar ferramentas de assistência de código
  • Desenvolver pensamento crítico e responsabilidade
  • Melhorar produtividade sem perder qualidade

Soft Skills na era da IA

Saber programar não é mais suficiente.
Saber decidir, validar e comunicar é essencial.

O novo papel do programador

  • Menos digitação
  • Mais análise
  • Mais revisão
  • Mais responsabilidade

IA como copiloto

  • Autocompleta código
  • Sugere soluções
  • Explica trechos
  • Gera arquivos inteiros

Ferramentas de IA para Código

  • GitHub Copilot
  • Cursor
  • Windsurf

GitHub Copilot

  • Integrado ao VS Code
  • Sugestões em tempo real
  • Baseado em contexto do projeto

Copilot – O que ele faz bem

  • Funções repetitivas
  • Boilerplate
  • Comentários → código

Cursor

  • Editor com IA nativa
  • Chat dentro do código
  • Refatoração assistida

Cursor – Diferencial

Entende o projeto inteiro, não só o arquivo atual.

Windsurf

  • Foco em fluxo de desenvolvimento
  • Geração de múltiplos arquivos
  • IA como parceira de arquitetura

Instalação no IDE

  • VS Code (extensões)
  • Login com conta
  • Configuração inicial

Integração com o projeto

Quanto melhor o projeto organizado,
melhor a sugestão da IA.

Autocompletação contextual

  • Nome de variáveis importa
  • Comentários ajudam
  • Estrutura influencia

IA não entende intenção

Ela prevê código provável.
Quem entende o problema é você.

Soft Skill: Pensamento Crítico

  • Isso faz sentido?
  • Está seguro?
  • Resolve o problema?

Validação do código gerado

  • Ler linha por linha
  • Testar entradas inválidas
  • Verificar segurança

Erro comum

Copiar código da IA sem entender.

Boas práticas

  • Use IA como rascunho
  • Refatore manualmente
  • Teste sempre
  • Documente

Soft Skill: Comunicação

Explicar código é tão importante quanto escrever código.

Soft Skill: Responsabilidade

Se o código falhar, a culpa não é da IA.

ATIVIDADE PRINCIPAL

Use uma ferramenta de IA para:
  • Gerar uma função
  • Revisar a função
  • Corrigir manualmente

Discussão em grupo

A IA ajudou ou atrapalhou?
Onde ela errou?

Reflexão Final

A IA acelera.
O humano decide.
