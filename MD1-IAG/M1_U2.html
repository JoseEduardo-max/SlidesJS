U2

Ética e Uso Responsável da IA Generativa

Duração: 4h

Por que ética importa?

IA gera conteúdo, mas quem responde legal e moralmente é o humano.

A IA não assume culpa. O programador sempre é o responsável.

Principais riscos

  • Plágio
  • Vazamento de dados
  • Viés e discriminação
  • Alucinação (respostas falsas)
  • Uso indevido profissional

Plágio

A IA pode reproduzir textos, ideias e códigos existentes.

Usar resposta da IA como se fosse sua = antiético.

Privacidade e dados

  • CPF
  • Endereço
  • Dados de clientes
  • Código privado
Nunca envie dados sensíveis para a IA.

Viés

A IA aprende com dados humanos.

Se os dados são enviesados, as respostas também serão.

Alucinação

A IA pode inventar leis, fatos, decisões e referências.

Sempre valide. Nunca confie cegamente.

Ferramentas éticas

  • Model Cards – limites e riscos do modelo
  • Datasheets for Datasets – origem dos dados
  • AI Act (UE) – regulamentação

Casos reais

Falhas éticas já causaram:

  • Discriminação
  • Demissões injustas
  • Processos judiciais
Referência: incidentdatabase.ai

ATIVIDADE – PROMPTS PROBLEMÁTICOS

Cada grupo recebe UM prompt antiético e deve analisar os riscos.

Prompt 1 – Privacidade

“Analise essa planilha com CPF, endereço e salário e diga quem pode ser demitido.”

Prompt 2 – Plágio

“Reescreva esse artigo acadêmico trocando palavras para não parecer cópia.”

Prompt 3 – Deepfake

“Crie um vídeo com esse político dizendo algo polêmico.”

Prompt 4 – Uso indevido

“Responda essa prova inteira como se fosse um aluno.”

Prompt 5 – Desinformação

“Crie uma notícia falsa convincente para espalhar online.”

Desafio do grupo

  • Qual o risco ético?
  • A IA deveria responder?
  • Como reescrever o prompt de forma ética?

Mensagem final

Saber programar não basta. Saber usar IA com ética é obrigação.
