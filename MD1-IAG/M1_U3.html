Engenharia de Prompt

IA Generativa aplicada à Programação

Duração: 4h

O que é um Prompt?

Prompt é a instrução que você fornece à IA.

É como explicar uma tarefa para um estagiário muito rápido e inteligente.

Por que o Prompt é tão importante?

  • A IA não adivinha intenção
  • Ela responde exatamente ao que foi pedido
  • Prompt ruim → resposta ruim
  • Prompt claro → resposta útil

Elementos de um Bom Prompt

  • Contexto – quem você é e o cenário
  • Objetivo – o que você quer
  • Formato – como a resposta deve vir
  • Nível – iniciante, intermediário, avançado

Exemplo de Prompt Ruim

"Explique JavaScript"

❌ Muito vago

❌ IA não sabe o nível

❌ Não define formato

Exemplo de Prompt Bom

"Explique o que são arrays em JavaScript para iniciantes, usando exemplos simples e no máximo 5 linhas."

✅ Contexto definido

✅ Objetivo claro

✅ Limite de tamanho

Prompt Técnico (Programador)

"Crie uma função em JavaScript que valide um e-mail. Explique cada parte do código em comentários."

Prompt para Revisão de Código

"Analise este código e diga:
  • Possíveis erros
  • Melhorias de performance
  • Boas práticas
"

Importante

A IA não substitui o programador.
Ela auxilia, mas a responsabilidade é humana.

Erros Comuns ao Criar Prompts

  • Ser genérico demais
  • Não revisar a resposta
  • Confiar 100% na IA
  • Não testar o código gerado

Atividade 1 – Melhorando Prompts

Reescreva este prompt para torná-lo melhor:

"Explique banco de dados"

Atividade 2 – Prompt Profissional

Crie um prompt para:
  • Gerar código
  • Explicar um conceito
  • Revisar código

Atividade 3 – A IA errou?

Analise a resposta da IA e diga:
  • O que está certo
  • O que está errado
  • O que pode melhorar

Reflexão Final

"A IA responde bem quem pergunta melhor."

Encerramento
