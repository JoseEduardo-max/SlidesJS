Exploração de APIs de LLMs

Duração: 4h

O que é uma API?

É uma interface que permite um sistema conversar com outro.

Como LLMs usam APIs

  • Aplicativo envia prompt
  • Modelo processa
  • Resposta é retornada

Conceitos importantes

  • Tokens
  • Temperatura
  • Contexto

Pseudocódigo

enviar(prompt) → API → LLM → resposta → aplicativo

Atividade

Interpretar um fluxo de chamada de API.
