Módulo 1

Introdução às APIs

O que é uma API?

API significa Interface de Programação de Aplicações.

É uma ponte que permite que sistemas diferentes se comuniquem.

APIs no dia a dia

  • Login com Google
  • Pagamentos online
  • Uber e Google Maps
  • Redes sociais

API não é interface gráfica

APIs não possuem botões ou telas.

APIs trabalham com dados (JSON).

Módulo 2

LLMs e APIs

O que é um LLM?

  • Large Language Model
  • Treinado com bilhões de textos
  • Especialista em linguagem

Como acessamos um LLM

Aplicações acessam LLMs através de APIs.

Fluxo básico

  • Usuário escreve
  • Aplicação envia o prompt
  • API processa
  • Resposta retorna

Módulo 3

Conceitos Técnicos

Tokens

Tokens são pedaços de texto usados pelo modelo.

"Olá, tudo bem?" → vários tokens

Temperatura

  • Baixa: respostas mais previsíveis
  • Alta: respostas mais criativas

Contexto

Contexto é o histórico da conversa.

Sem contexto, o modelo não lembra do que foi dito antes.

Módulo 4

Fluxo de uma API

Pseudocódigo

enviar(prompt) → API → LLM → resposta → aplicativo

Exemplo conceitual

"Explique o que é uma API para iniciantes"

Problemas comuns

  • Prompt mal formulado
  • Contexto muito longo
  • Limite de tokens
