Introdução à IA Generativa e LLMs

Duração: 4 horas

Base do módulo de IA Generativa

Por que essa aula é importante?

A IA generativa muda a forma como software, texto, imagens e código são criados.

Não é uma ferramenta. É uma nova forma de trabalhar.

IA tradicional vs IA Generativa

  • IA tradicional → classifica, detecta, decide
  • IA generativa → cria conteúdo novo
  • Ela não busca respostas, ela gera respostas

O que a IA Generativa pode criar?

  • Textos
  • Código
  • Imagens
  • Vídeos
  • Áudio e voz

O que é um LLM?

LLM = Large Language Model

Um modelo treinado com bilhões de palavras para prever a próxima palavra.

Como um LLM funciona?

  • Ele lê um texto (prompt)
  • Quebra em tokens
  • Calcula probabilidades
  • Gera a próxima palavra
  • Repete o processo

LLM não pensa

Ele não entende como humanos.

Ele prevê padrões com base em dados gigantescos.

ChatGPT não é a IA

  • ChatGPT = interface
  • LLM = cérebro
  • API = forma de acesso

Exemplos de LLMs

  • GPT-4 / GPT-5
  • Claude
  • LLaMA
  • Gemini

IA Generativa e Programadores

O programador deixa de ser apenas quem escreve código.

Ele passa a ser quem orienta uma IA a criar soluções.

O novo fluxo de desenvolvimento

  • Você descreve o problema
  • A IA gera o código
  • Você valida, corrige e integra

IA como copiloto

  • Explica código
  • Gera funções
  • Ajuda a debugar
  • Traduz lógica

Limitações dos LLMs

  • Podem errar
  • Podem inventar respostas
  • Dependem do prompt

Por que isso é revolucionário?

Pela primeira vez, qualquer pessoa pode conversar com um sistema que escreve, cria e programa.

Atividade prática

Peça para um LLM:

  • Gerar um texto
  • Gerar um código
  • Explicar um conceito

Reflexão

Quem usar IA trabalhará melhor do que quem não usar.
